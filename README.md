## 📱 Featured iOS Projects

### [SlangSpeak](https://apps.apple.com)

Educational iOS app for learning Brazilian slang through interactive and gamified experiences.

* Built with SwiftUI, MVVM, StoreKit, App Intents, CloudKit, and Firebase Cloud Messaging.
* Implemented CloudKit persistence for saved slang entries and user-generated content.
* Integrated Firebase Cloud Messaging with APNs to support Push Notifications.
* Built a premium subscription flow with StoreKit.
* Added App Intents to support Siri voice interactions.

### [PinSpot](https://apps.apple.com)

Native iOS app for saving and organizing places using interactive maps.

* Built with SwiftUI and MapKit.
* Implemented map interactions, location-based UI, dynamic navigation, and SwiftUI animations.
* Shipped to the App Store and reached 150+ organic downloads in the first two months.

### [AR Strike](https://apps.apple.com)

Augmented reality iOS app focused on immersive multiplayer HUD interfaces.

* Built with SwiftUI, ARKit, Firebase Cloud Messaging, and Localizable.strings.
* Implemented Push Notifications with Firebase Cloud Messaging.
* Created a reusable SwiftUI Design System for HUD components.
* Added multilingual support using Localizable.strings.

### [Fimbo Village](https://www.youtube.com/watch?v=b-l3RBeFTzg)

Interactive iOS experience for learning music theory.

* WWDC 2025 Swift Student Challenge Distinguished Winner.
* Selected among the Top 50 projects worldwide.
* Built with a focus on accessibility, interaction design, and native Apple platform features.
